jQuery Rbox Demo and Documentation

To use:
Include jQuery and the rbox script:

Include jQuery, followed by the jquery-rbox script in the head tag or just below the closing body tag.

        
            <script src="http://code.jquery.com/jquery-1.11.0.min.js"></script>
            <script src="jquery-rbox.js"></script>
Include the stylesheet:

Include the jquery-rbox.css stylesheet in the head tag

        
            <link rel="stylesheet" href="jquery-rbox.css" /glt;
Call rbox on an element:

e.g. something with a class of foo

Do this below the rbox script tag

        
            $(document.ready(function() {
                $('.foo').rbox();
            });

Features

  • Responsive across devices
  • Can be used with inline elements, single images and videos, image and video galleries, iframes and ajax.
  • Really lightweight, unminified is 12kb!

Options

Name Type Default Description
'series' string series this lightbox is a part of
'type' string image, video, inline, html, iframe or ajax
'image' path to image, for type image
'video' path to video file, for type video
'iframe' path to iframe, for type iframe
'ajax' URL to fetch ajax content from
'html' add some html here
'inline' selector for element on page whose innerHTML is the content
'caption' optional caption
'fade' 300 default? fade delay: maybe do this via css?
'width' Integer?? 0 is this for the iframe or something?
'height' Integer?? 0 is this for the iframe or something?
'videoposter' poster image path for videos
'autoplay' false if type==video, if video should autoplay
'fitvid' false whether to use the fitvids.js plugin (needs to be included). This ensures responsive videos
'namespace': namespace, 'loading': 'Loading...', 'beforeopen': function(opts) { return opts; }, //called before open 'onopen': function() { $.noop(); }, //called onopen 'onclose': function() { $.noop(); }, //called onclose 'beforeclose': function() { $.noop(); }
